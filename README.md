# 📚 लाइब्रेरी मैनेजमेंट सिस्टम (Library Management System)

यह पाइथन (Python) में बना एक सरल और प्रभावी प्रोजेक्ट है। इसमें **Object-Oriented Programming (OOPs)** का उपयोग किया गया है ताकि कोड साफ़, व्यवस्थित और समझने में आसान रहे।

---

## 🛠️ प्रोजेक्ट की विशेषताएँ (Features)

* **OOPs आधारित संरचना:** `Library` क्लास का उपयोग करके डेटा और फंक्शन को व्यवस्थित किया गया है।
* **किताबें जोड़ना (Add Books):** यूज़र रन-टाइम पर नई किताबों के नाम दर्ज कर सकता है।
* **लाइब्रेरी देखें (View Library):** जोड़ी गई सभी किताबों की सूची को एक अच्छे फॉर्मेट में देख सकते हैं।
* **ऑटो-काउंटर:** प्रोग्राम अपने आप बताता है कि लाइब्रेरी में कुल कितनी किताबें मौजूद हैं।
* **स्मार्ट लूप:** `while` लूप और `lower()` फंक्शन का उपयोग करके यूज़र इंटरफ़ेस को सरल बनाया गया है।

---

## 💻 कोड कैसे काम करता है? (How it Works)

1. **Class `Library`:** यह ब्लूप्रिंट है जिसमें किताबों की लिस्ट और उनकी संख्या को मैनेज किया जाता है।
2. **`add_book` मेथड:** यूज़र से इनपुट लेकर किताब को `self.book` लिस्ट में जोड़ता है।
3. **`view_labrbry` मेथड:** `for` लूप की मदद से लिस्ट में मौजूद हर किताब को प्रिंट करता है।
4. **Main Loop:** प्रोग्राम तब तक चलता रहता है जब तक यूज़र 'no' नहीं टाइप कर देता।

---

## 🚀 शुरू कैसे करें (Setup & Run)

1. अपने कंप्यूटर पर **Python** इंस्टॉल करें।
2. इस कोड को `library.py` नाम की फ़ाइल में सेव करें।
3. टर्मिनल या कमांड प्रॉम्प्ट में नीचे दिया गया कमांड चलाएँ:

```bash
python library.py
```
